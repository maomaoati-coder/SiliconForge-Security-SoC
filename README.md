## 核心模块

- **RO-PUF** — 64路环形振荡器，生成芯片唯一物理指纹
- **Bio-Hash** — ARX轻量哈希引擎，128bit身份令牌验证
- **异常检测FSM** — 实时指令监控，纳秒级硬件拦截
- **Security SoC** — 三模块全链路整合顶层
